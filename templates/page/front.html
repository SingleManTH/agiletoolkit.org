Agile Toolkit is a simple PHP framework for developing rich Web User Interface.

Sleek, intuitive, and powerful front-end framework for faster and easier web development.

Download Agile Toolkit 4.3
See it in action

UI Design

Agile Toolkit comes with a powerful HTML-based, templates and many UI widgets you can use such as Forms, Buttons, Sliders, Menus as a standard.

Community Hub

While we all work on our own web apps, Agile Toolkit connects us together in a global way letting us share resources, knowledge and help.

Easy to Learn

A powerful framework that is easy to learn. We work hard to make sure Agile Toolkit could become your first programming environment.
